Partner App → GIS

APIs called from partner-app-service to GIS for loan registration and processing

# Name Method URL Description Link
1 Check Duplicate Customer POST /kbpapp/api/checkDuplicateCustomer Checks the active status of existing contracts (duplicate) based on email or mobile View
2 Pre Approval (RL) POST /kbpapp/api/preApproval Pre-approval for RL loan application View
3 Customer Info Send POST /kbpapp/api/customerInfoSend Sends customer personal information for loan processing View
4 Employment Info Send POST /kbpapp/api/employmentInfoSend Sends customer employment information for loan processing View
5 Loan Application POST /kbpapp/api/loanApplication Submits a loan application with all required details View
6 RLML Loan Application POST /kbpapp/api/rlmlLoan Submits an RLML loan application View
7 Document Info Send POST /kbpapp/api/DocumentInfoSend Sends document file paths for loan processing View
8 NCB Data POST /kbpapp/api/ncbData Sends NCB credit bureau data for evaluation View
9 Cancel Loan POST /kbpapp/api/loanCancel Cancels an existing loan application View
10 Check Application Status POST /kbpapp/api/checkApplicationStatus Checks current status of a loan application View
11 RLML Info Inquiry POST /kbpapp/api/rlmlInfo Retrieves RLML loan information for a customer View
12 EMI Inquiry POST /kbpapp/api/emiInquiry Calculates EMI for RLML loans View
13 Pickup POST /kbpapp/api/pickup Updates pickup status for RLML loan View
14 Project Inquiry POST /kbpapp/api/projectInquiry Retrieves available project information (cached once per day per OA) View
15 Promotion Inquiry POST /kbpapp/api/promotionInquiry Retrieves available promotion information (cached once per day per OA) View
